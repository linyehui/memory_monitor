# 项目规格说明书：内存监控器 (Memory Monitor)

## 1. 概述
一个轻量级的命令行工具，用于实时监控系统内存使用情况，记录数据，并在使用率超过定义阈值时发出警报。

## 2. 目标
- 实时监控 RAM 和 Swap 使用情况。
- 命令行第一个参数可以是一个进程ID，用于指定要监控的内存大小

- 命令行从运行开始持续在控制台中分行显示以下几个信息：
  * 命令行刚运行是的进程内存情况:RAM、Swap
  * 当前时刻的进程内存情况:RAM、Swap
  * 从命令行运行到当前的内存增量情况
- 命令行中的显示不要持续换行，而是在固定的行里面不断刷新
- 当内存达到关键阈值时，控制台上的增量内存信息使用红色显示
- 监控工具本身应保持低资源占用

## 3. 技术栈 (建议)
- **语言**: 使用macOS系统自带的shell脚本，不要依赖其他第三方库或语言
