# 项目技能清单 (Skills)

本项目旨在提供以下核心能力：

## 1. 实时监控 (Monitoring)
- **能力描述**: 能够以指定的时间间隔（默认 1 秒）读取系统内存状态。
- **输入**: 采样间隔 (秒)。
- **输出**: 包含总内存、已用内存、空闲内存、使用百分比的结构化数据。

## 2. 数据持久化 (Logging)
- **能力描述**: 将监控到的数据流式写入本地文件。
- **支持格式**: CSV (便于表格分析), JSON (便于程序处理)。
- **策略**: 支持日志轮转 (Log Rotation) 防止文件过大。

## 3. 智能告警 (Alerting)
- **能力描述**: 持续评估当前内存状态是否违反预设规则。
- **规则类型**: 
    - 阈值告警 (如：使用率 > 90%)
    - 趋势告警 (如：1分钟内增长 50%) - *高级功能*
- **反馈方式**: 终端高亮输出、桌面通知 (可选)。

## 4. 命令行交互 (CLI Interaction)
- **能力描述**: 提供友好的命令行界面供用户配置和控制。
- **功能**: 启动/停止监控、设置阈值、指定日志文件路径。
